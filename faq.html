Frequenty Asked Questions

How may I suggest a feature?

Suggesting features is done by opening a PR (Pull Request) on GitHub

Could you implement X feature?

I will implement every suggestions as long as it follows the following rules:

  • Is part of the "basics" of mongodb/mongoose.
  • Is easily implementable and adapts easily with the current framework.
  • Has a good reason to be implemented.
