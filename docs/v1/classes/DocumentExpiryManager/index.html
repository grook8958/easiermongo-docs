Properties

.model

The model bound to this manager.
Type: MongoMode

.clockPRIVATE

The interval clock.
Type: NodeJS.Timer

.willExpire

An array of Ids of documents who will expire.
Type: Array <String >

Methods

.checkExpire()

Check if a document is about to expire. (And removes it)
Returns: void

.destroy()

Destroy this manager.
Returns: void

.register(id)

Register a new document to the manager.
PARAMETER TYPE DESCRIPTION
id string The unique Identifier of the document.
Returns: boolean

.remove(id)

Remove a document from the manager.
PARAMETER TYPE DESCRIPTION
id string The unique Identifier of the document.
Returns: boolean

Events

expire

Emitted when a model registered in the manager expires.
PARAMETER TYPE DESCRIPTION
id string The model ID.
document MongoDocument The document who just expired.
