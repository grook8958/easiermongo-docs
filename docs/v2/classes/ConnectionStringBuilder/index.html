ConnectionStringBuilder

The object representation of a MongoDB connection string.

Constructor

new ConnectionStringBuilder(data);
PARAMETER TYPE DESCRIPTION
data ConnectionStringBuilderData The connection string data.
  • host
  • port
  • dbName
  • username
  • password
  • srv
  • authSource
  • setHost
  • setPort
  • setDbName
  • setUsername
  • setPassword
  • useSRV
  • setAuthSource
  • toString
  • toJSON

Properties

.host

The host address.
Type: string

.port

The port number.
Type: number

.dbName

The database name.
Type: string

.username

The connection username.
Type: string

.password

The connection password.
Type: string

.srv

Wether it should use mongodb+srv://. Set to false by default
Type: boolean

.authSource

The Authentication Source.
Type: string
