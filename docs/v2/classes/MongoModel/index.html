MongoModel

The representation of a model.

PARAMETER TYPE OPTIONAL DEFAULT DESCRIPTION
model Model none A mongoose model.

Properties

._model

The model it is representing
Type: Model

.makeCache

Wether easiermongo should cache documents when fetched from the database.
Type: boolean

.cache

Wether easiermongo should cache documents when fetched from the database.
Type: Collection

.ttl

The TTL (Time-To-Live) a document in this collection has.
Type: number |null

._expiryManager

The expiry manager of this model
Type: DocumentExpiryManager

Methods

.create(document)

Create a new document in the collection.
PARAMETER TYPE DESCRIPTION
document DocumentBuilder|BaseDocumentData The document object according to what was
set when creating the schema.
Returns: Promise <MongoDocument>

.getAll()

Get all the documents in this collection.
Returns: Promise <Array <MongoDocument>>

.get(id)

Get a document by it's id.
PARAMETER TYPE DESCRIPTION
id string |ObjectId The document _id
Returns: Promise <MongoDocument>

Example: 
const document = await model.get('someId')
  .catch(err => console.error(err))
console.log(document)

.find(query)

Searches for one document matching the query

WARNING

If you want to find a document using it's _id use the get method instead

PARAMETER TYPE DESCRIPTION
query MongoQuery The query to search for
Returns: Promise <MongoDocument>

.findMany(query)

Searches for all document matching the query
PARAMETER TYPE DESCRIPTION
query MongoQuery The query to search for
Returns: Promise <Array <MongoDocument>>

.edit(id, change, [options])

Edits a document using it's id.

TIP

Setting the option new to true returns the newly edited document!

PARAMETER TYPE OPTIONAL DESCRIPTION
id string The document's _id
change MongoChange The changes it should apply to the document
options ModelEditOptions The options of this edit
Returns: Promise <MongoDocument>

Example: 
const newDocument = await model.edit('some-id', { age: 48 }, { new: true })
 .catch(err => console.error(err))
console.log(newDocument);

.findAndEdit(query, change, [options])

PARAMETER TYPE OPTIONAL DESCRIPTION

.editMany(query, change, [options])

Finds and edits one or more documents
PARAMETER TYPE OPTIONAL DESCRIPTION
query MongoQuery The query to search for
change MongoChange The changes that should apply to all the documents matching the query

.delete(id)

Deletes a document using it's id.
PARAMETER TYPE DESCRIPTION
id string The document's _id
Returns: Promise <void >

Example: 
model.edit('some-id')
 .then(() => console.log('Deleted document of id "some-id"'))
 .catch(err => console.error(err))

.findAndDelete(query)

Finds a document then deletes it.
PARAMETER TYPE DESCRIPTION
query MongoQuery The query to search for
Returns: Promise <void >

.deleteMany(query)

Finds documents and deletes them
PARAMETER TYPE DESCRIPTION
query MongoQuery The query to search for
Returns: Promise <void >

.update(id, change, [options])

Updates a document without overriding content of objects.

DANGER

This behaves like .edit() except it doesn't override content of objects.
Decreased performance are to expect
IMPORTANT: This will not work unless updating objects

PARAMETER TYPE OPTIONAL DESCRIPTION
id string The document's _id
change MongoChange The changes it should apply to the document
options ModelEditOptions The options of this update
Returns: Promise <MongoDocument>

.findAndUpdate(query, change, [options])

.updateMany(query, change, [options])

.onExpire(callback)

Runs the callback each time a document expires.

WARNING

Documents created before the code was ran are not taken into notice

PARAMETER TYPE DESCRIPTION
callback Function <string , MongoDocument> The function to run when a document expires
Returns: void
