Properties

.dbREAD-ONLY

The database to which a connection was opened.
Type: Database

.database

The database to which a connection was opened.
Type: Database

.options

The options that were used to initialize this client.
Type: MongoClientOptions

.uriREAD-ONLY

The URI connection string.
Type: ?ConnectionStringBuilder|?string

._mongoose

The native mongoose connection to the database.
Type: Mongoose

Methods

.connect(uri)

Opens a new connnection to your database.
PARAMETER TYPE DESCRIPTION
uri ConnectionStringBuilder|string The options to use for your client.
Returns: Promise <Database>

.disconnect()

Close the connection from your database.
Returns: Promise <void >

Events

error

Emitted when an error occurs.
PARAMETER TYPE DESCRIPTION
error Error The error thrown.

disconnected

Emitted when then connection gets disconnected.

disconnecting

Emitted when the connection is disconnecting.

connected

Emitted when the connection is established.
PARAMETER TYPE DESCRIPTION
database Database The database where the connection was opened.

connecting

Emitted when a connection is trying to be established.

reconnected

Emitted when the connection is reconnected after being disconnected.

close

Emitted when the connection is sucessfully closed.

ready

Emitted when the client is ready to perform actions.

file

Emitted when a schema file is registered.
PARAMETER TYPE DESCRIPTION
file string The schema file name.
